What we can do with object model(DOM)

JS gets all the power it need to create dynamic HTML
  1. JS can change all the HTML elements in the page
  2. Js can change all the HTML element attributes in the page
  3. Js can change all the CSS style in the page
  4. JS can add new HTML element and attributes
  5. JS can revome existing HTML elements and attributes
  6. JS can react to all existing HTML events in the page
  7. JS can create new HTML event in the page

Step: Find HTML Element

We know that DOM (Document) objects has every meta data about webpage.
  1. document.getElementByID(ID); Find element by ID
  2. document.getElementByTagName(name); Find element by tagname;
  3. document.getElementByClassName(ClassName); Find element by ClassName
  4. document.querySelector(Selector); #elementID, .Classname, tagname

Step2: Change HTML elements:

    Property:
  1. element.InnerHTML = new html content ==> change the html content
  2. element.InnerText = new html content
  3. element.attribute = new value ==> change the attribute value of an HTML element
  4. element.style.property = new style ==> change the style of html element
    5. Method: element.setAttribute(attribute, value) ==> Add an attribute to the html element

Step3: Adding and Deleting ELements:

  1. document.createElement(Element): create an html element
  2. document.removeChild(element): Remove an HTML element
  3. document.apendChild(element): add an HTML element
  4. document.replaceChild(new, old): replace an element
